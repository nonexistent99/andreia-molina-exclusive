🚀 COMO TESTAR
─────────────────────────────────────────────────────────────────────────────
1. Configure as variáveis de ambiente (.env.local):
   - DATABASE_URL (MySQL)
   - LXPAY_API_KEY e LXPAY_API_SECRET
   - BREVO_API_KEY
   - JWT_SECRET

2. Instale dependências:
   pnpm install

3. Execute migrations:
   pnpm db:push

4. Inicie o servidor:
   pnpm dev

5. Acesse http://localhost:5173
   - Clique em "Comprar" em qualquer produto
   - Sistema criará pedido automaticamente
   - Será redirecionado para página de pagamento com QR code
